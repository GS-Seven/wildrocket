# <center>wildrocket
### <center>简介
把项目中常用的场景封装成一个包，便于新项目的快速开发

### 安装
```shell
go get github.com/jeffcil/wildrocket
```

&#10004; 常用加密 md5、sh1、文件md5、aes

&#10004; jwt认证

&#10004; map操作  两个map合并

&#10004; redis客户端初始化

&#10004; mysql客户端初始化、连接池

&#10004; orm (gorm、xorm)

&#10004; 动态搜索+分页 （xorm）
